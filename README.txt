REQUEST HEADER PARSER MICROSERVICE

OVERVIEW
This is a simple Node.js and Express-based microservice that parses incoming 
HTTP request headers to extract key client information. 
It returns the user's IP address, language preferences, and browser information in a JSON format.

FEATURES
    •    Parses and returns the client’s IP address

    •    Detects and returns the preferred language from headers

    •    Retrieves the user-agent string to identify browser and operating system

PREREQUISITES
    •    Node.js and npm installed

INSTALLATION
    •    Clone this repository

    •    Navigate into the project folder

    •    Run npm install to install dependencies

    •    Start the server with npm start or node index.js

PROJECT STRUCTURE
    •    index.js – Main application file with Express server and route

    •    package.json – Project metadata and dependencies
